## ✨ Features

- 🎬 Fetch latest official trailers
- 🚫 Filters reaction & fake videos
- 📅 Shows only recent uploads
- 📺 Supports multiple studios
- 📤 Send directly to Telegram channel
- ⚡ Fast & stable (no timeout issues)

---

## 🧠 How It Works

- Searches YouTube for trailers
- Filters:
  - Official channels
  - Removes junk videos
  - Keeps recent uploads
- Displays list to select
- Sends to your Telegram channel
